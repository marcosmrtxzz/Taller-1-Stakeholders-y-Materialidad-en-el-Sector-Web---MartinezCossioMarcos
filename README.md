# Stakeholders y Materialidad en el Sector Web

En este repositorio tenemos la tarea completa del Taller 1 del módulo.
El objetivo de este Taller 1 es saber cuales son los grupos de interes, los temas materiales ASG y hacer las estructuras XML Y XSD
##  1. Stakeholders Identificados

Los **tres grupos de interés clave** para una consultora web son:

- **ClientesUsuarios**  
  Impactados por la accesibilidad, la privacidad y el rendimiento.

- **Equipo interno de desarrollo**  
  Afectado por diversidad, ética en IA y cultura organizacional.

- **Proveedores de infraestructura cloud**  
  Determinan la huella de carbono y la gobernanza del dato.

##  2. Temas Materiales ASG

- **Eficiencia energética del código** (*Ambiental*)  
- **Accesibilidad universal e inclusión digital** (*Social*)  
- **Privacidad y gobernanza del dato** (*Gobernanza*)

##  3. Archivos del Repositorio

- `materialidad.xml` → Estructura de stakeholders y temas materiales.  
- `README.md` → Documentación profesional del taller.
